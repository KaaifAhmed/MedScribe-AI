Instant Reports

How It Works

A look under the hood at the AI pipeline powering MedScribe.

1. Input

Audio is captured via the browser's MediaRecorder API or file upload.

2. Analyze

FastAPI backend uploads audio to Gemini 1.5 Flash for multimodal processing.

3. Extract

The LLM diarizes speakers and extracts SOAP components from mixed-language speech.

4. Report

Structured JSON is rendered into a UI and converted to a downloadable PDF.
